1. Назначение
Telegram-бот для помощи в формировании привычек. Позволяет добавлять привычки, отмечать выполнение, смотреть статистику и получать советы от ИИ. Работает локально через aiogram 3 + SQLite.

2. Технологии
Python 3.11+ + aiogram 3 (бота)
SQLite (хранение данных)
OpenAI API через OpenRouter (советы от ИИ)
python-dotenv (настройки через .env)

3. Структура
text
project/
├── main.py                    # запуск бота
├── bot/handlers.py           # обработчики сообщений
├── database/manager.py       # работа с БД
├── database/models.py        # модели User, Habit
├── ai/agent.py              # интеграция с ИИ
├── config/settings.py       # загрузка настроек
└── .env                     # токены и пути

4. База данных
users (user_id, username, first_name, created_at)
habits (id, user_id, name, period, created_at)
entries (id, habit_id, date, done, note, created_at)
Таблицы создаются автоматически при первом запуске.

5. Функционал
Основное:
    /start — регистрация, главное меню
    Добавить привычку — название + периодичность
    Мои привычки — список с ID
    Отметить выполнение — выбор по номеру, запись в БД
    Статистика — количество выполнений по каждой привычке
ИИ-фича:
    Совет от ИИ — выбор привычки → 3-5 практических советов от AI
Техническое:
    Все данные в локальном SQLite-файле
    ИИ через OpenRouter (можно отключить)
    Кнопочный интерфейс в Telegram
    Обработка ошибок и /cancel

6. Как работает
Пользователь пишет /start → получает меню кнопок
Добавляет привычку → она сохраняется в БД
Отмечает выполнение → запись с датой
Смотрит статистику → подсчёт выполнений
Запрашивает совет → ИИ анализирует привычку → даёт рекомендации

7. Особенности
Простота: не нужно устанавливать приложение
Приватность: все данные хранятся локально
AI-помощь: умные советы по привычкам
Масштабируемость: можно добавить напоминания, экспорт, цели

8. Запуск
Установить зависимости: pip install aiogram python-dotenv openai
Запустить: python main.py
Написать боту в Telegram(@habit_tracker_misisbot)
